  2. 使用 .head()、.tail()、.sample() 方法，观察、了解数据表的情况。
  3. 使用 .dtypes 属性，来查看各列数据的数据类型。
  4. 使用 isnull() 配合 .any() 等方法，来查看各列是否存在空值。
  5. 使用 .describe() 方法，看看数据表中数值型的数据是怎么分布的。
